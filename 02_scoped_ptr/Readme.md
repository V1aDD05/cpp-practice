Код является **переработанным** решением [задания из курса](https://stepik.org/lesson/4650/step/3?unit=1055)

## Состав директории
- Readme
- Заголовочный файл `scoped_ptr.hpp` с реализацией шаблона `ScopedPtr`;
- Файл `example.cpp` с тестом методов шаблона.

## Идея
Связать время жизни динамического объекта с объектом на стеке – в деструкторе `ScopedPtr` вызывается `delete`.

## Особенности
- Копирование запрещено (удалены конструктор копирования и оператор присваивания).
- Поддержка перемещения (конструктор перемещения и оператор присваивания перемещением) позволяет передавать владение.

## Интерфейс
- `get()` — возвращает сырой указатель;
- `release()` — передаёт владение, `ScopedPtr` больше не удаляет объект;
- `reset` — удаляет старый объект и захватывает новый;
- `operator*()` и `operator->()` — доступ к объекту.
