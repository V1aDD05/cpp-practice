# Учебная низкоуровневая реализация нешаблонного класса целочисленного массива IntArray
Код является **переработанным** описанием [класса целочисленного массива из курса](https://stepik.org/lesson/551/step/1?unit=874)

## Состав директории
- Readme
- Заголовочный файл `int_array.hpp` с объявлением полей и методов класса `IntArray`;
- Файл `int_array.cpp` с описанием полей и методов класса `IntArray`;
- Файл `example.cpp` с тестом методов класса.

## Особенности
Класс выполнен нешаблонным для демонстрации навыка работы с раздельной компиляцией. Заголовочный файл защищён от повторного включения при помощи `include guards`.
Оператор копирования реализован в соответствии с идиомой `copy-and-swap`

## Интерфейс
- `swap()` - обмен данными двух объектов класса;
- `size()` - получение размера массива;
- `int get(size_t) const` - чтение элемента массива
- `int& get(size_t)` - доступ к элементу массива по ссылке
- `void resize(size_t nsize)` - изменение размеров массива

## Пример применения
Пример применения подробно показан в файле `example.cpp`.

g++ -std=c++17 example.cpp int_array.cpp -o test && ./test
