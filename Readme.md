Учебные реализации ряда сущностей и концепций на `C++` для демонстрации понимания управления памятью, RAII, правила трёх/пяти, перегрузки операторов и наследования.

Приведена реализация:
- динамического массива (директория `array`);
- умных указателей (директория `scoped_ptr`);
- задачи на применение наследования (директория `inheritance`);
- задачи на применение виртуальных методов (директория `virtual methods`);
- задачи на перегрузку операторов (директория `operator overload`).

Некоторые задачи выполнены в рамках [курса по C++ от Computer Science Center (Stepik)](https://stepik.org/7) с последующей доработкой.
[Сертификат](https://stepik.org/cert/3106085), подтвержающий прохождение курса.
