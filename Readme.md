Учебные реализации ряда сущностей и концепций на `C++` для демонстрации понимания управления памятью, RAII, правила трёх/пяти, перегрузки операторов и наследования.

## Приведена реализация:
- **динамического массива** в виде *нешаблонного* класса `IntArray` (директория `01_array`);
- **умных указателей** в виде *шаблонного* класса `ScopedPtr` (директория `02_scoped_ptr`);
- задачи на применение **наследования и виртуальных методов** (директория `03_inheritance_polymorphism`);
- задачи на **перегрузку операторов** (директория `04_operator_overload`), для этого проекта реализовано тестирование при помощи **Google Test**, для автоматизации сборки применяется **CMake**.

## Технические особенности
- все заголовочные файлы защищены от повторного включения при помощи `include guards`;
- обработка исключений не используется (учебный проект);
- для проекта по перегрузке операторов используется **Google Test** и **CMake** – как пример внедрения современных инструментов тестирования
