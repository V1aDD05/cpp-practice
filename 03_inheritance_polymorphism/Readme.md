# Иерархия классов для представления арифметических выражений
Код является **переработанным** решением [задания из курса](https://stepik.org/lesson/554/step/8?unit=877)

## Состав директории
- Readme
- Заголовочный файл `expression.hpp`, содержащий объявление базового класса `Expression` иерархии классов;
- Заголовочный файл `number.hpp`, содержащий объявление класса представления чисел`Number`;
- Заголовочный файл `binary_operation.hpp`, содержащий объявление класса представления бинарных операций (`+`, `-`, `*`, `/`)`BinaryOperation`;
- Файл `example.cpp` с тестом методов иерархии классов.

## Особенности
- Базовый абстрактный класс `Expression` с чисто виртуальной функцией `evaluate()`.
- Класс `Number` – лист дерева выражения. Копируемый и перемещаемый
- Класс `BinaryOperation` – узел с двумя операндами (левым и правым) и операцией. Некопируемый (из-за `std::unique_ptr`), но перемещаемый.
- Управление памятью через `std::unique_ptr` – автоматическое освобождение.

## Интерфейс
- `double evaluate() const` - вычисляет значение `double` арифметического выражения

## Пример применения
Пример применения подробно показан в файле `example.cpp`.

g++ -std=c++17 example.cpp -o example && ./example
